第3章 程序逻辑循环

3.1 分支与循环

分支与循环是程序逻辑的基本控制单元，存在三种程序逻辑：顺序结构、分支结构、循环结构。

顺序结构：对于之前所编写的代码都是顺序结构，即所有程序代码的顺序依次按顺序自上至下执行。

分支结构（if 结构）：if 分支结构主要是针对关系表达式进行判断处理的分支操作。对于分支语句有三种组成形式：if 判断：if(…){}、if else判断：if(…){} else {}、多条件判断：if(…){} else if(…){} [else{}] 。

在使用 if 的时候最主要的特点是它可以进行若干条件的判断，在 Java 中只有 if else 语句可以实现多条件判断。在进行多条件判断时，可以不写else语句，但好的习惯是一定要写上else语句。

switch 中有个重要的语句：break;
在使用switch语句时，如果你在每一个case后面没有追加break语句，那么会在第一个匹配的case之后继续执行其他语句，直到遇到 break; ，或全部 switch 代码执行完毕。

在 JDK 1.7 版本，Oracle 公司对 Java 添加了 switch 对字符串 String 类型数据的判断。

switch 语句是一个编程语言发展中的一种重要技术，但进入桌面开发时代后 switch 语句出现频率变得很低。
