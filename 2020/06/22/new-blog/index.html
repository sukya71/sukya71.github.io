  4. 初始化博客

  5. 启动博客

  6. 安装 git deployer 插件

    此步为了将博客部署在 github 远端上。

    安装 Git Bash（https://gitforwindows.org/），在其中安装 git deployer。

    Git Bash:

    编辑 _config.yml 文件下方找到 Deployment 修改 repo 地址。

    Deployment

    Docs: https://hexo.io/docs/deployment.html

    deploy:
    type: git
    repo: https://github.com/(git仓库地址)
    branch: master

    生成博客：

    此时在 github 仓库网页上刷新会发现出现很多文件。

    输入 github 仓库网址即可浏览博客。

    更换主题示例

    主题地址：https://github.com/litten/hexo-theme-yilia

  8. 应用

    编辑 _config.yml

    Extensions

    Themes: https://hexo.io/themes/

    theme: yilia
