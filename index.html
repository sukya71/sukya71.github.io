# 第3章 程序逻辑循环

第3章 程序逻辑循环

3.1 分支与循环

分支与循环是程序逻辑的基本控制单元，存在三种程序逻辑：顺序结构、分支结构、循环结构。

顺序结构：对于之前所编写的代码都是顺序结构，即所有程序代码的顺序依次按顺序自上至下执行。

分支结构（if 结构）：if 分支结构主要是针对关系表达式进行判断处理的分支操作。对于分支语句有三种组成形式：if 判断：if(…){}、if else判断：if(…){} else {}、多条件判断：if(…){} else if(…){} [else{}] 。

# 第2章 程序概念基础

第2章 程序基础概念

2.1 程序注释

程序注释是项目开发中的一项重要组成技术，合理的注释会让项目开发更加的方便，但很多公司技术管理不到位，很多注释不写的，所以就造成了人员更换时期的痛苦。

编译器在进行程序编译的时候发现有注释的内容将不对此部分进行编译，注释分三类形式。

  • 单行注释：// ……

  • 多行注释：/* ……*/

  • 文档注释：/** ……*/，文档注释里还需要很多的选项，一般建议通过开发工具控制。

# 第1章 走进Java的世界

第1章 走进Java的世界

1.1 Java发展历史

Java可以实现任何需求，有些时候Java不擅长的部分会被其他语言所替代。国内而言Java使用广泛的原因是有很多的大户都在使用Java实现其各自的核心业务。

1991年 GREEN项目，智能家居，SUN公司打算竞标此项目，考虑到C++语言开发的复杂性，于是基于C++语言开发出了一套自己的独立平台Oak（橡树），并且用此平台开发的项目竞标，可惜失败了。

这时候世界上最早的软件公司诞生了：网景（Netscape），受到了当时浏览器的启发，SUN公司工程师开发出了HotJava浏览器。1995年5月23日推出了Java语言及JDK1.0开发包，1996年才开放下载与使用。1997年传到中国。
